# Stopwatch Web Application

## Overview

This project is a modern and interactive Stopwatch Web Application developed as part of the SkillCraft Technology Web Development Internship (Task 02).

The application allows users to accurately measure and record time intervals using stopwatch functionality. It includes features such as start, pause, reset, lap recording, theme switching, lap export, and a responsive user interface.

## Features

* Start Stopwatch
* Pause Stopwatch
* Reset Stopwatch
* Record Multiple Lap Times
* Export Lap Records to Text File
* Dark / Light Theme Toggle
* Animated Circular Progress Ring
* Responsive Design for Mobile and Desktop
* Keyboard Shortcuts Support
* Modern User-Friendly Interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## How to Use

1. Click **Start** to begin timing.
2. Click **Pause** to stop the timer temporarily.
3. Click **Lap** to save the current time as a lap record.
4. Click **Reset** to clear the timer and lap records.
5. Click **Export** to download lap records as a text file.
6. Use the theme toggle button to switch between dark and light modes.
